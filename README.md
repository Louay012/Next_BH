# NEXT by BH Assurance — AI Multi-Task Sales Agent

## 📌 Description

This project was developed in the context of the **NEXT by BH Assurance challenge**.

### Challenge objective

Build a **multi-task AI agent** capable of:

- Analyzing the customer profile
- Detecting business opportunities
- Generating a persuasive pitch and sending it through the best communication channel
- Following and supporting the negotiation process with the customer

---

## 🎯 Project Goals

The solution aims to help insurance/commercial teams:

- Better understand each client
- Personalize offers and recommendations
- Improve conversion rates with targeted outreach
- Track interactions and adapt strategy during negotiations

---

## 🧠 Core Features

- **Client Profile Analysis**
  - Collects and structures client data
  - Builds actionable customer insights

- **Opportunity Detection**
  - Identifies upsell/cross-sell opportunities
  - Scores potential commercial actions

- **Persuasive Pitch Generation**
  - Creates customized, context-aware sales messages
  - Adapts tone and value proposition to the customer profile

- **Best Channel Selection**
  - Recommends and/or triggers the most relevant channel (e.g. email, message, call workflow)

- **Negotiation Follow-Up**
  - Tracks customer responses and status
  - Suggests next best actions during the negotiation lifecycle

## 🚀 Getting Started

> Adjust commands according to your actual project structure.

### 1) Clone the repository

```bash
git clone https://github.com/Louay012/Next_BH.git
cd Next_BH
```

### 2) Backend setup (Python)

```bash
python -m venv .venv
source .venv/bin/activate  # On Windows: .venv\Scripts\activate
pip install -r requirements.txt
```

Run backend (example):

```bash
python app.py
```

### 3) Frontend setup (JavaScript)

```bash
npm install
npm run dev
```
